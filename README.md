# ここにゲームのタイトルを記述する
ROLL FOR SCORE
## ゲームの内容
本作『ROLL FOR SCORE』は、リアルタイム進行で展開される1対1の対戦ゲームである。プレイヤーは生成されるサイコロを自分の盤面に配置し、制限時間内に高いスコアを獲得することを目指す。
ゲーム画面は1画面固定で構成され、画面の上下にプレイヤーと対戦相手それぞれの 3×3の盤面 が配置されている。
各盤面は3本の列で構成されており、サイコロは好きなマスに配置できる。
プレイヤーはサイコロを1つだけ所持することができる。
サイコロを盤面に配置すると、その出目の数値が得点として加算される。
さらに、同じ列に同じ数値のサイコロが複数揃うと、その列の得点倍率が上昇する。例(N＋N)×２　(N＋N＋N)×３
ゲームは時間が止まることなく進行し、プレイヤーと対戦相手は同時に行動する。
制限時間終了時、またはお互いの盤面がすべて埋まった時点でゲームが終了し、CPUより最終スコアが高いプレイヤーが勝利となる。
また、プレイヤーはリロールや相手の行動を制限するスキルを使用することができ、スキルには使用回数が設定されている。
これにより、盤面状況やタイミングを見極めた戦略的な判断も重要な要素となっている。
本作は、短時間で決着がつく中に、瞬時の判断による駆け引きを詰め込んだリアルタイム対戦ゲーム。
